# 📊 Customer Churn Analysis (Python + Power BI)

## 📌 Project Overview

This project focuses on analyzing and predicting customer churn using **Python (Machine Learning)** and visualizing insights using **Power BI Dashboard**.

The goal is to identify customers likely to churn and provide actionable insights to improve customer retention.

---

## 🎯 Objectives

* Predict customer churn using machine learning
* Analyze customer behavior and patterns
* Build an interactive dashboard for business insights
* Help businesses reduce customer churn

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Scikit-learn)
* Power BI
* Power Query
* DAX
* Machine Learning Algorithms

---

## 📂 Dataset Features

* CustomerID
* Gender
* Age
* Tenure
* Contract Type
* Subscription Type
* Monthly Charges / Total Spend
* Support Calls
* Usage Frequency
* Churn (Target Variable)

---

## 🧠 Python (Machine Learning)

### 🔹 Data Preprocessing

* Handled missing values
* Encoded categorical variables
* Scaled numerical features

### 🔹 Model Building

* Trained classification model (e.g., Logistic Regression / Random Forest)
* Evaluated using accuracy and confusion matrix

### 🔹 Outcome

* Predicted whether a customer will churn or not

---

## 📊 Power BI Dashboard

### 🔹 KPIs

* Total Customers
* Churn Customers
* Churn Rate (%)

### 🔹 Visualizations

* Churn by Contract Type
* Churn by Gender
* Churn by Subscription Type

### 🔹 Slicers

* Gender
* Contract Type
* Subscription Type

---

## 📈 Key Insights

* Customers with **monthly contracts** show higher churn
* **Basic subscription users** are more likely to churn
* **Gender has minimal impact** on churn

---

## 💼 Business Recommendations

* Encourage customers to move to long-term contracts
* Improve value in basic subscription plans
* Focus retention strategies on high-risk customers
